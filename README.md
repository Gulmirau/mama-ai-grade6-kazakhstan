# Mama AI

MVP web-приложения Mama AI - учебного AI-компаньона для учеников 6 класса по школьной программе Казахстана.

## Как открыть

1. Откройте папку `mama_ai_app`.
2. Запустите файл `index.html` двойным кликом.
3. Приложение откроется в браузере без установки.

## Что реализовано по PRD

- Пошаговый чат-помощник, который объясняет тему, а не просто дает ответ.
- Выбор предметов: математика, английский, естествознание, казахский язык, подготовка к СОР и СОЧ.
- Расширенная предметная база для 6 класса: русский язык, русская литература, казахская литература, история Казахстана, всемирная история, география, информатика, художественный труд.
- Развернутые ответы по ключевым темам: определение, объяснение, пример, план оформления ответа и вопрос для самопроверки.
- Переключатель языка объяснения: RU, KZ, EN.
- Прототип загрузки фото задания.
- Ежедневный учебный план.
- Мини-тест после урока с начислением баллов.
- Геймификация: баллы, серия дней, уровень, поддерживающие ответы.
- Панель родителя с прогрессом по предметам и рекомендацией.

## Текущий статус

Это фронтенд-прототип без настоящего подключения к AI API и без реального распознавания фото. Логика ответов пока встроена в `script.js`, чтобы показать продуктовый сценарий. Для полностью корректных ответов по любому вопросу из школьной программы следующим этапом нужно подключить AI API и официальные учебные материалы/КТП школы.
